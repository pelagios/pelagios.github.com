How to add an icon with a link to a Place Widget

May blog and content management systems do not allow you to embed scripts. Here are instructions for adding a Pelagios icon with a link to a particular place widget as an alternative.

1) Find the Pleiades ID of the place

Find the place on the Pleiades website and look at the URL of the page for the place to get the Pleiades ID of the place. e.g. if the URL is http://pleiades.stoa.org/places/579885/ or http://pleiades.stoa.org/places/579885/?searchterm=Athens* then the Pleiades ID is 579885.

2) Add the following HTML to your page where PLEIADES_ID is the Pleiades ID from the first step: 

<a href="http://pelagios.github.com/pelagios-widgets/demo/place/place.html?pleiadesid=PLEIADES_ID">
<img src="http://pelagios.github.com/pelagios-widgets/images/icons/pelagios.png" /></a>
